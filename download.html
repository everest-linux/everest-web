Download Everest

Most users will want to get the most recent release.

If for some reason you require an older release,

it may be in the release archive, which contains all

deprecated releases.

Using a very old release is not recommended as many critical

programs will likely have security vulnerabilities.

ALWAYS verify checksums of system images.

There are multiple ways of getting Everest:

HTTPS

BitTorrent

In some regions, using BitTorrent is illegal.

Please ensure accessing BitTorrent is safe to do.

Everest Linux assumes no responsibility.

If using a web seed capable client, please leave it open once finished,

in order to seed back.

wget

If you want to quickly download a system image, you can use wget.

(user)$ wget https://git.everestlinux.org/EverestLinux/everest/raw/branch/main/releases/SYSTEM_IMAGE
